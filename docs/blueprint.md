# **App Name**: CuraLink

## Core Features:

- Patient Profile Creation with AI: Patients create profiles by inputting medical details in natural language; AI then identifies conditions, and starts populating relevant results.
- Personalized Dashboard: Display a personalized dashboard recommending publications, health experts, and clinical trials based on the patient's condition.
- Clinical Trial Search with AI Summaries: Enable users to search clinical trials using keywords and filters; include AI-generated summaries of the trials.
- Expert Connection: Allow patients to search and follow health experts, and request meetings; for non-registered experts, pull public data.
- Researcher Profile Creation & Publication Auto-Import: Allow researchers to create profiles with specialties and research interests, with optional links to ORCID/ResearchGate for auto-importing publications. Display publications with AI generated summaries. Also, include an option for researchers to indicate availability for meetings.
- Forums with Expert Replies: Create forums for discussions; researchers can create communities and answer questions. Patients may not reply to each other, and may not create new communities.
- Favorites: Patients and researchers can save their favorite publications, clinical trials, and experts/collaborators.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6), reflecting trustworthiness and health.
- Background color: Light grey (#F0F4F8), to ensure content stands out.
- Accent color: Soft green (#A5D6A7), complementing the blue and giving a sense of growth.
- Headline font: 'Poppins', a sans-serif with a modern feel.
- Body font: 'PT Sans', a slightly warmer sans-serif; to be used with 'Poppins' which is recommended for headings.
- Use clean and modern icons for navigation and features.
- Clean and minimal layout, similar to Duolingo, with clear sections for each feature.
